# Styla

Run the Styla REPL

```
$ sbt "runMain prolog.Main"
```

The Prolog prompt should appear

```
Styla: Scala Terms Yield Logic Agents
Prolog System Version 1.3.4 with LogicActors
Copyright (C) Paul Tarau 2011-2014
?-
```


Test by typing

```
?- member(3, [1,2,3]).
yes.
no (more) answers
```
